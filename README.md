# вҡҷ Getting Started

## Prerequisites

* Node.js (v18+)
* npm or yarn

---

## Clone Project

```bash
git clone https://github.com/your-username/techhub.git
cd frontend
```

---

## Install Dependencies

```bash
npm install
```

---

## Environment Setup

Create `.env` file:

```env
VITE_API_URL=http://your-api-url
VITE_GOOGLE_CLIENT_ID=your-google-client-id
VITE_GITHUB_CLIENT_ID=your-github-client-id
```

---

## Run Development

```bash
npm run dev
```

App runs at:

```
http://localhost:5173
```

---

## Build Production

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
